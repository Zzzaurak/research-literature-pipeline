# Project Start Prompt

Use this when starting a new literature project.

## Input

- Research topic:
- Target output:
- Field / subfield:
- Must include:
- Must exclude:
- Time range:

## Task

1. Restate the research question in one paragraph.
2. Generate search terms and synonyms.
3. Identify 3-5 likely method families.
4. Identify seed papers or authors if known.
5. Propose Zotero collection name and tags.
6. Create a candidate search plan.

## Output

- Project summary
- Search query list
- Screening rubric
- Initial Zotero tags
- Next actions
